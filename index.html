Introduction Regional summary

This page summarises the number of diagnosed coronavirus (covid-19) cases recorded daily by Public Health England (PHE).

The PHE dashboard presents the number of cases, recovered patients, and deaths nationally as well as daily cumulative counts of cases for NHS Regions and Upper Tier Local Authorities and Unitary Authorities.

Data collection started on the 9th March 2020.

This page works best in certain browsers such as Chrome, Firefox, and Safari, and does not work so well currently in Internet Explorer.
